# Thread Project
## Overview
This project is a wearable free fall detection system implemented on an ESP32. It uses ESP_IDF. 
## Repo Structure
The documentation folder contains a user guide, diagrams, and pictures. 
The enclosure folder contains .stl files for 3D printing the enclosure. 
The code folder contains 2 subfolders. The final firmware is in the node_final folder.
The final ui, as well as some helper Python programs, are in the data_proc folder. See code/data_proc/requirements.txt for the requirements for setting up the venv.
