# Thread Project
## Overview
This project is a wearable free fall detection system implemented on an ESP32. It uses ESP_IDF. 
## Repo Structure
The documentation folder contains a user guide, diagrams, and pictures. 
The enclosure folder contains .stl files for 3D printing the enclosure. 
The code folder contains 5 subfolders. the ones with "test" in the name are just examples of using
each component separately (CSI, microphone, vibration sensor, etc.). The final code is in the node_final folder.
The final ui, as well as some helper Python programs, are in the data_proc folder.
